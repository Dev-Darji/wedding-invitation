# Aarav & Ishita's Digital Kankotri 🪔

A premium, interactive digital wedding invitation website built with traditional Indian aesthetics and modern web animations.

## Features
* 🎀 **Interactive Knot Opening:** A physical-feeling "untie the knot" (Naada/Dori) bowtie mechanism to reveal the invitation.
* 🏰 **Indian Royal Theme:** Features a premium color palette (Maroon, Gold, and Cream) with traditional patterns (Mandala and Paisley).
* ⏳ **Real-Time Countdown:** Keeps track of the days, hours, minutes, and seconds until the celebration starts.
* 📜 **Overlapping Event Stack:** A premium vertical scrolling layout where scheduled function cards dynamically slide over each other.
* 🎵 **Shehnai Background Music:** Traditional music that plays automatically on open, complete with interactive ambient sounds.
* 💌 **WhatsApp RSVP Integration:** Built-in form that synthesizes guest responses and sends them formatted over WhatsApp.
* 📅 **Calendar & PDF Downloads:** Direct integration to save the event to Google Calendar/Apple Calendar (.ics) and download a print-friendly invitation.

## Technologies Used
* HTML5
* Vanilla CSS3
* Custom SVG Art
* JavaScript (ES6+)
* Canvas API (Party Popper Confetti)
* Web Audio API (Synthesized Temple Bells)
